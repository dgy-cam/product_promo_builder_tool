Optional Details

Product handles looks like:

  • sari-wrap-skirt
  • darn-good-yarn-bowl
  • lace-weight-silk-yarn

They're lowercase, always uses dashes, and can be found at the end of the URL for the product after the last slash.

Like: https://darngoodyarn.com/products/sari-wrap-skirt

Product or Variant IDs looks like:

  • 14502170545024
  • 2052347502348502540
  • 4592475820435702

They're numbers, vary in length, and can be found at the end of the URL for the product in shopify admin.

Like: https://darn-good-yarn-2.myshopify.com/admin/products/7536931270/variants/29817672646

The 1st number is the Product ID.

The 2nd is a Variant ID, which only shows up when you click on a variant in the admin as if you were trying to edit it.

If a product doesn't have any variants, you'll leave the variant ID field blank.
